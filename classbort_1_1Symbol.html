Detailed Description

Some program entity representation

Symbol is a representation of any (i quess) named entitiy in a program (function, variable, typedef, ...). The key concept in symbol resolution is symbol shallowness: shallow symbols are just names by which actual symbol information will be retrieved later (in resolution process).
