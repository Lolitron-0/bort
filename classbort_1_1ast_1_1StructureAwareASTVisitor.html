Detailed Description

Base class of all SA visitors.

Structure aware visitors act more like an AST pattern matching, except they are a bit more low-level as they can perform operation anywhere during the tree traversal. By default, non-overriden nodes simply continue traversal with no action. Pre- and post-order visitors can be easily implemented (and even mixed) using this interface
